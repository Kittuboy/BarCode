Generate QR Code

1. Enter text or URL in the input field

2. Select "QR Code" type

3. Click "Generate" button

4. Download or save the generated QR code

Generate Barcode

1. Enter text in the input field

2. Select "Barcode" type

3. Click "Generate" button

4. Download or save the generated barcode

Scan Codes

1. Click the "Scan" tab

2. Allow camera access when prompted

3. Point your camera at a QR code or barcode

4. The result will appear automatically
